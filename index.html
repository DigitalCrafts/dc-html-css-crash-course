How the Web Works

It's All in the Browser

What's the Browser Doing?

  1. Request the website (ex: google.com)
  2. Find server using Domain Name System (DNS)
  3. Download the files from the server
  4. Process the files
  5. Show the result to the user
  6. Repeat
